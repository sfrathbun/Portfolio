REACT JS

Neighboring Application for the COVID-19 Global Church Hackathon.

COVID-19 Global Church Hackathon Team Project -April 2020

Since the Corona virus outbreak in the beginning of 2020, Faith Tech put on a Hackathon challenge. With 10 challenges to choose from and one week until completion, our team of 12 developers set our faces like flint to create a website that would serve our neighbors with their basic needs. A user can create a profile and request food and household items that they are lacking. The local Church would then step in to provide for the needs of the community. There is also a way to give in order to help supply the needs of others as well.

The front end was build with REACT JS using react-bootstrap. The website includes login, password hashing, JWT, user and admin profile views. The back end was built with EXPRESS JS so that users can post their needs to a database. All available items are stored in the database as well as the user and admin profile information. I'm so grateful to be a part of this wonderful team.

My contributions to this project

As a part of the front end team, I mostly helped with creating the footer, the give page, the profile page, and some of the home page. I built the home page slider and created the images for it, as well as most of the other images throughout the site. We had team standup meetings everyday to regroup and share our progress. Because this was a one week Hackathon challenge, we worked tirelessly as a team, communicating remotely through zoom and used GitHub to source control all of our pushes to the master branch. It was truly an amazing effort by the entire team in the short amount of time we had.

EXPRESS JS Social Media App

Express JS, Handlebars JS, and MySQL database

Social Media App using MySQL database

This is a fully functioning social media app that was built for a Bethel Tech school project. It allows users to sign up, login, and create posts. This app uses salt and password hashing for extra security measures. JSON web tokens are used to allow the user to navigate through the app without being logged out. The back end was built with EXPRESS JS so that users can post to the database. All posts are stored in the database as well as the user and admin profile information. Certain users may also have admin privileges, giving them access to private information, also allowing them to delete posts and users from the database.

Angular 9 Task Application

Angular 8 with JSON Mock Server

Task App using Angular 8 and TypeScript

This Angular 8 Task App was built for a Bethel Tech school project with Angular 8. The core of this app uses CRUD operators in order to create, read, update, and delete tasks. Angular page routing is super fast, giving a great user experience with no page load wait time. The app uses a HTTP request with a JSON mock in order to send and receive the data from page to page. Some simple CSS styling was included.
